# How to USE
1. Create Google Docs
2. Copy Article
3. Use Script (create new) goto `Tools > Scripts Editor`
4. Paste Script 2 files and save them
5. In Script Editor Goto `Publish > Publish to web apllication` and Click ` Get lastest code`
6. Fin
